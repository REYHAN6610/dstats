> *Dstat is a system monitoring tool for tracking server performance (CPU, memory, network). Its function is to detect DDoS attacks by monitoring traffic spikes, not to perform attacks.*

![Dstat Demo](https://qu.ax/CaNPM.gif)

## Features
- ✅ Monitor CPU, memory, and network usage
- ✅ Detect abnormal traffic patterns (DDoS detection)

## Quick Start

```bash
git clone https://github.com/REYHAN6610/dstats
cd dstats
pkg install cloudflare -y

yarn install #or npm install
npm start
```

*Note:* This tool is for system monitoring only, not for hacking purposes.
